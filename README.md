# Student Management System

A simple and efficient Student Management System developed using Python and SQLite database. This project helps manage student records through a menu-driven console application.

---

## Features

- Add new student records
- View all students
- Search students by name
- Update student grades
- Delete student records
- SQLite database integration
- Object-Oriented Programming concepts

---

## Technologies Used

- Python
- SQLite3
- DBMS
- OOP Concepts

## How to Run the Project

1. Open the project folder in VS Code

2. Open terminal

3. Run the following command:

```bash
python main.py
```

---

## Functionalities

### Add Student
Stores student details such as:
- Name
- Age
- Email
- Course
- Grade

### View Students
Displays all student records in tabular format.

### Search Student
Search student details using student name.

### Update Grade
Modify grades of existing students.

### Delete Student
Remove student records from the database.
